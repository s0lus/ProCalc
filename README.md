# Calculator

### Functionality
* Linear equation (ax + b = c)
* Quadratic equation (ax^2 + bx + c = 0)
* Squaring (x^2)
* Cubed (x^3)
* x^y [y - int]
* Factorial (n!)

### Install
>> mkdir build && cd build <br>
>> cmake .. <br>
>> make <br>
>> ./ProCalc <br>
